Spotify Song Popularity Analysis

Analyzed Spotify song data to understand which audio features are most related to song popularity.

Python pandas matplotlib Jupyter Notebooks
More details

Problem

With so many different audio features available for songs, it can be difficult to understand what actually makes a song popular. The goal of this project was to identify which features have the strongest relationship with popularity.

Approach

I analyzed a dataset of Spotify songs using Python and pandas. I explored relationships between features such as energy, danceability, loudness, tempo, and valence using summary statistics and visualizations to identify patterns.

Results & Impact

The analysis showed that energy and danceability had strong positive relationships with popularity, while tempo and loudness had little effect. This helps highlight which characteristics are more important when predicting song success.

Spartan Football Crowd Cheer Analysis

Analyzed audio and attendance data from Spartan Stadium to understand what game events drive the loudest crowd reactions. Results were used to help the athletics department optimize in-game entertainment and — personally — to determine the exact moments when I should run onto the field for maximum dramatic effect.

Python pandas seaborn Plotly Jupyter Notebooks
More details

Problem

Fan energy is contagious, but it is also unpredictable. The athletics department wanted to know which in-game events (touchdowns, turnovers, big defensive stops, walk-up music choices) produced the biggest crowd reactions — and whether the pattern differed by opponent, weather, or game time. As the mascot, I had strong professional interest in this question.

Approach

Combined play-by-play game data with decibel readings from stadium microphones across three seasons (~36 home games). Engineered features for score differential, down-and-distance, quarter, temperature, and opponent rank. Produced interactive Plotly dashboards that let non-technical staff filter by season and event type to explore crowd response patterns on their own.

Results & Impact

Defensive turnovers produced the loudest average crowd response — louder even than touchdowns — especially when MSU was trailing. Walk-up music selections in the 3rd quarter showed a statistically significant effect on cheer volume for games decided by fewer than 10 points. The athletics team used these findings to redesign the third-quarter entertainment package for the following season. I was given extra treats for my contributions.
